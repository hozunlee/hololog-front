# Project Overview: front-hololog

front-hololog is the frontend application for the Hololog platform. It is a modern web application built using SvelteKit and TailwindCSS.

## Key Features
- **Blog Platform:** Likely supports creating, reading, and managing blog posts.
- **Markdown Support:** Uses `svelte-markdown` for rendering content.
- **Backend Integration:** Connects to Supabase for data persistence and authentication.
- **Responsive Design:** Utilizes TailwindCSS for styling.
- **Icons:** Uses `lucide-svelte` for iconography.

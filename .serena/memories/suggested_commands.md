# Suggested Commands

## Development
- `npm run dev`: Start the development server.
- `npm run build`: Build the application for production.
- `npm run preview`: Preview the production build locally.

## Quality Assurance
- `npm run check`: Run SvelteKit synchronization and `svelte-check` (TypeScript/JSDoc checking).
- `npm run check:watch`: Run `svelte-check` in watch mode.
- `npm run lint`: Run Prettier check and ESLint to ensure code quality.

## Formatting
- `npm run format`: Automatically format code using Prettier.

## System Utilities (Darwin)
- `ls -la`: List files with details.
- `grep -r "search_term" .`: Recursive search for text.
- `find . -name "filename"`: Find files by name.

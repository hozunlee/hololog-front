# Tech Stack

## Core
- **Framework:** SvelteKit
- **UI Library:** Svelte 4
- **Language:** JavaScript (with JSDoc typing)
- **Build Tool:** Vite

## Styling
- **CSS Framework:** TailwindCSS
- **Icons:** Lucide Svelte
- **UI Components:** Bits UI

## Backend & Data
- **Backend as a Service:** Supabase (`@supabase/supabase-js`)
- **HTTP Client:** Axios
- **Date Handling:** Dayjs

## Deployment
- **Platform:** Vercel (using `@sveltejs/adapter-vercel`)

## Dev Tools
- **Linting:** ESLint
- **Formatting:** Prettier
- **Type Checking:** Svelte Check (TypeScript)
